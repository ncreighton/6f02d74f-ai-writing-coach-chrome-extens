# AI Writing Coach Chrome Extension

Chrome extension providing AI-powered writing feedback in any text field. Analyses tone, clarity, reading level, and grammar as you type. Offers one-click rewrites for passive voice, wordy phrases, and jargon. Features a sidebar with Hemingway-style readability score, word count targets, and custom style guide enforcement. Supports academic, professional, and casual writing modes. Works in Gmail, Google Docs, LinkedIn, Twitter, and any web form with no copy-paste required. Powered by local Ollama for privacy-first users with optional cloud fallback.

## Installation

### Chrome
1. Download and extract the ZIP
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extracted folder

### Firefox
1. Download the Firefox version
2. Go to `about:debugging`
3. Click "Load Temporary Add-on"
